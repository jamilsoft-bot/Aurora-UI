# Demos

Aurora includes HTML demos in the `demos/` directory. You can either open files directly (no-build) or run a local server.

## No-build demo
1) Build CSS:
   ```bash
   npm run build:css
   ```
2) Open `demos/basic/index.html` in your browser.
3) Open `demos/layout/index.html` to explore layout utilities.

## Local server demo
```bash
npm run demo
```
Then visit:
- http://localhost:5173/demos/basic/
- http://localhost:5173/demos/layout/
- http://localhost:5173/demos/components/
- http://localhost:5173/demos/templates/landing.html
- http://localhost:5173/demos/templates/dashboard.html
