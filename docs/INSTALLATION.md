# Installation

This guide walks you through installing dependencies and generating the CSS build output.

## Prerequisites
- **Node.js 18+** (Node 20/22 works too)
- **npm** (bundled with Node)

Check your versions:
```bash
node -v
npm -v
```

## Install dependencies
```bash
npm install
```

## Build the CSS output
```bash
npm run build:css
```
Output:
- `dist/styles.css`
